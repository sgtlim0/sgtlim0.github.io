# 검색엔진 AI 카드

H Chat v3의 검색엔진 AI 카드 기능은 Google, Bing, Naver 검색 결과 페이지에 자동으로 AI 요약 카드를 주입합니다.

## 주요 기능

### 자동 AI 요약
- 검색 쿼리 자동 감지
- 검색어에 대한 AI 답변 즉시 생성
- 검색 결과 상단에 요약 카드 표시

### 스마트 UI
- **접을 수 있는 카드 UI**: 필요에 따라 카드를 접거나 펼칠 수 있습니다
- **전체 채팅 열기**: 카드에서 바로 전체 채팅 인터페이스로 이동
- **Shadow DOM 격리**: 검색 페이지 스타일과 충돌하지 않는 독립적인 스타일링

### 지원 검색엔진
- Google 검색
- Bing 검색
- Naver 검색

## 작동 방식

1. 사용자가 검색엔진에서 검색을 수행합니다
2. H Chat이 검색 쿼리를 자동으로 감지합니다
3. AI가 검색어에 대한 답변을 생성합니다
4. 검색 결과 상단에 요약 카드가 자동으로 삽입됩니다
5. 사용자는 카드를 접거나 펼치거나, 전체 채팅으로 이동할 수 있습니다

## 기술 구현

이 기능은 `src/content/search-injector.ts`에 구현되어 있으며, Shadow DOM을 사용하여 검색 페이지의 기존 스타일과 격리된 독립적인 UI를 제공합니다.

## 설정

검색엔진 AI 카드는 기본적으로 활성화되어 있습니다. 설정 페이지에서 이 기능을 끄거나 켤 수 있습니다.
