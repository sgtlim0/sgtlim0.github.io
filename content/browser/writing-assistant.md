# 글쓰기 어시스턴트

H Chat v3의 글쓰기 어시스턴트는 웹페이지의 모든 textarea에 플로팅 툴바를 추가하여 실시간 AI 텍스트 변환 기능을 제공합니다.

## 주요 기능

### 플로팅 툴바
- 모든 웹페이지의 textarea에 자동으로 나타나는 플로팅 툴바
- 텍스트 입력 중 언제든지 접근 가능
- 비침투적인 UI 디자인

### 7가지 AI 변환

1. **개선 (Improve)**: 텍스트의 가독성과 명확성을 향상시킵니다
2. **축약 (Shorten)**: 핵심 내용을 유지하면서 간결하게 만듭니다
3. **확장 (Expand)**: 더 자세하고 풍부한 내용으로 확장합니다
4. **전문적 (Professional)**: 비즈니스 및 공식 문서에 적합한 톤으로 변환합니다
5. **캐주얼 (Casual)**: 친근하고 편안한 톤으로 변환합니다
6. **교정 (Proofread)**: 맞춤법, 문법, 구두점 오류를 수정합니다
7. **번역 (Translate)**: 다른 언어로 번역합니다

## 작동 방식

1. 웹페이지의 textarea에 텍스트를 입력합니다
2. 플로팅 툴바가 자동으로 나타납니다
3. 원하는 변환 옵션을 선택합니다
4. AI가 실시간으로 텍스트를 변환합니다
5. 변환된 결과가 즉시 textarea에 적용됩니다

## 사용 예시

### 이메일 작성
- 초안을 작성한 후 "전문적" 옵션으로 비즈니스 이메일에 적합한 톤으로 변환
- 긴 이메일은 "축약" 옵션으로 핵심만 간결하게 정리

### 소셜 미디어
- 공식적인 텍스트를 "캐주얼" 옵션으로 친근한 톤으로 변환
- 짧은 글을 "확장" 옵션으로 더 풍부한 내용으로 만들기

### 문서 작성
- 작성 중인 문서를 "개선" 옵션으로 가독성 향상
- 완성된 문서를 "교정" 옵션으로 최종 검토

## 기술 구현

이 기능은 `src/content/writing-assistant.ts`에 구현되어 있으며, 모든 textarea 요소를 감지하고 플로팅 툴바를 동적으로 추가합니다.

## 지원 사이트

글쓰기 어시스턴트는 다음을 포함한 모든 웹사이트에서 작동합니다:
- Gmail, Outlook 등 이메일 서비스
- Twitter, LinkedIn 등 소셜 미디어
- Notion, Google Docs 등 문서 편집 도구
- GitHub, GitLab 등 개발 플랫폼
- 기타 textarea를 사용하는 모든 웹페이지
