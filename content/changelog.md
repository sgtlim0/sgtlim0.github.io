# 변경 이력

## v3.0.0 (2026년 3월) - 멀티 AI 프로바이더 시스템

H Chat v3의 가장 큰 업데이트로, 단일 AI 프로바이더에서 멀티 프로바이더 아키텍처로 전환했습니다.

### 🎯 신규 기능

#### 멀티 AI 프로바이더 지원

- **OpenAI GPT 프로바이더 추가**
  - GPT-4o, GPT-4o mini 지원
  - OpenAI API 직접 통합
  - 스트리밍 응답 지원

- **Google Gemini 프로바이더 추가**
  - Gemini Flash 2.0, Gemini Pro 1.5 지원
  - Google AI Studio API 통합
  - 멀티모달 입력 지원

- **총 7개 AI 모델 지원**
  - AWS Bedrock: Claude Sonnet 4.6, Claude Opus 4.6, Claude Haiku 4.5
  - OpenAI: GPT-4o, GPT-4o mini
  - Google: Gemini Flash 2.0, Gemini Pro 1.5

#### 자동 모델 라우팅

- 작업 유형 자동 감지
- 최적의 AI 모델 자동 선택
- 코드 분석, 텍스트 요약, 창의적 글쓰기 등 작업별 최적화
- 사용자 커스터마이징 가능

#### 크로스 모델 토론

- 서로 다른 프로바이더의 AI 모델 간 대화
- 동시 다중 모델 응답 비교
- 모델 간 의견 차이 시각화
- 최대 4개 모델 동시 토론 지원 (최소 2개)

#### YouTube 댓글 분석

- YouTube 동영상 댓글 크롤링
- AI 기반 감정 분석
- 주요 의견 요약
- 주제별 댓글 분류

#### PDF 채팅

- PDF 파일 업로드 및 텍스트 추출
- PDF 내용 기반 질의응답
- 다국어 PDF 지원
- 특정 페이지 참조 가능

#### 검색 엔진 AI 카드

- Google 검색 결과에 AI 요약 추가
- Bing, Naver 검색 엔진 지원
- 실시간 웹 컨텍스트 통합
- 출처 링크 제공

#### 글쓰기 어시스턴트

- 텍스트 입력창에서 AI 글쓰기 지원
- 문법 교정 및 개선 제안
- 스타일 변환 (격식체/비격식체)
- 다국어 번역

#### 프로바이더별 사용량 추적

- 프로바이더별 토큰 사용량 통계
- API 호출 횟수 추적
- 예상 비용 계산
- 모델별 사용 패턴 분석

### 🏗️ 아키텍처 개선

#### Provider Factory 패턴

- 통합된 프로바이더 인터페이스
- 새 프로바이더 추가 용이
- 타입 안전성 강화
- 의존성 주입 지원

#### useProvider 훅

- React 커스텀 훅으로 프로바이더 관리
- 프로바이더 전환 간소화
- 상태 관리 최적화
- 재사용성 향상

#### 향상된 메시지 흐름

- 스트리밍 응답 최적화
- 에러 핸들링 개선
- 재시도 로직 추가
- 타임아웃 관리

### 📊 프로젝트 규모

- **코드 라인**: 10,000+
- **탭 (기능)**: 8개

- **AI 프로바이더**: 3개

- **기능**: 21개

### 🔧 개선 사항

- **그룹 채팅**: 크로스 프로바이더 비교 지원
- **성능**: 응답 속도 30% 개선
- **UI/UX**: 프로바이더 전환 UI 개선
- **설정**: 프로바이더별 설정 분리
- **저장소**: 효율적인 데이터 구조
- **보안**: API 키 암호화 강화

### 🐛 버그 수정

- 긴 대화에서 메모리 누수 수정
- 스트리밍 중단 시 에러 처리 개선
- 다국어 입력 처리 버그 수정
- PDF 추출 인코딩 문제 해결

## v2.0.0 (2025년 12월) - 웹 검색 + 멀티턴 에이전트

### 🎯 신규 기능

#### 웹 검색 + RAG

- **DuckDuckGo HTML 스크래핑** + **Google CSE** 통합
- 검색 결과 1시간 캐시
- RAG (Retrieval-Augmented Generation) 구현
- 검색 결과 요약 및 출처 참조
- 최신 정보 기반 응답

#### 멀티턴 에이전트

- **XML 기반 도구 호출** 시스템
- 최대 10스텝 에이전트 실행
- **5개 내장 도구**: 웹 검색, 페이지 분석, 북마크 조회, 하이라이트 조회, 대화 검색
- 복잡한 작업 자동 분해 및 단계별 실행
- 작업 진행 상황 실시간 추적

#### 스마트 북마크

- **텍스트 하이라이팅** + XPath 영속화
- **5가지 색상** 하이라이트 지원
- AI 기반 북마크 요약 및 태깅
- 북마크 검색 및 카테고리 관리
- 하이라이트한 텍스트 문맥 보존

#### 페이지 컨텍스트

- 현재 페이지 내용 자동 분석 (최대 8,000자)
- 시맨틱 요소 추출 (제목, 단락, 링크)
- 페이지 구조 파악 및 주요 정보 추출
- 컨텍스트 기반 질의응답

#### 플로팅 AI 툴바

- 텍스트 선택 시 **7가지 AI 액션** 메뉴
- **설명하기**, **번역하기**, **요약하기**, **다듬기**, **격식체로**, **교정**, **하이라이트**
- 선택 텍스트 즉시 처리
- 결과를 사이드패널에 자동 표시

#### 그룹 채팅

- 동일 프로바이더 내 모델 비교 (Claude 3개 모델)
- 동시 다중 응답 표시
- 모델 간 응답 차이 비교

#### 도구 패널

- **7가지 AI 도구** 통합 인터페이스
- 페이지 요약, 전체 번역, 텍스트 선택 처리 등
- 빠른 접근 UI

#### 프롬프트 라이브러리

- 기본 **8개 프롬프트** 제공
- 카테고리 필터링 (일반/코딩/글쓰기)
- 커스텀 프롬프트 추가/편집

#### 대화 기록 관리

- 대화 검색, 태그, 고정 기능
- 대화 가져오기/내보내기
- 대화별 메타데이터 관리

#### 키보드 단축키 시스템

- 주요 기능 빠른 접근
- 사용자 지정 단축키
- 단축키 가이드 제공

#### TTS/STT 지원

- 텍스트 음성 변환 (Text-to-Speech)
- 음성 텍스트 변환 (Speech-to-Text)
- 다국어 음성 지원

### 📊 프로젝트 규모

- **소스 파일**: 35개+
- **코드 라인**: 7,000+
- **기능**: 15개
- **AI 모델**: Claude Sonnet 4.6, Claude Opus 4.6, Claude Haiku 4.5

### 🔧 개선 사항

- **채팅 UI**: 메시지 렌더링 최적화
- **사이드패널**: 반응형 레이아웃
- **설정**: 고급 옵션 추가
- **에러 처리**: 네트워크 오류 복구
- **저장소**: chrome.storage.local 최적화

### 🐛 버그 수정

- 사이드패널이 간헐적으로 닫히는 문제 수정
- AWS 인증 만료 처리 개선
- 긴 응답 렌더링 버그 수정
- 메모리 사용량 최적화

### ⚠️ 제거된 기능 (v1 → v2)

v1에서 제공되던 다음 기능들이 v2에서 제거되었으며, 웹 검색, 멀티턴 에이전트 등 새로운 기능으로 대체되었습니다:

- **대화 메모리** (CLAUDE.md 스타일 메모리)
- **작업 스케줄러** (chrome.alarms 기반)
- **에이전트 스웜** (병렬 에이전트 실행)
- **FAB** (Floating Action Button)
- **팝업 미니 대시보드**

## v1.0.0 (2024년 6월) - 최초 릴리스

### 🎯 초기 기능

#### Chrome 확장 프로그램

- Manifest V3 기반
- **5탭 사이드패널 UI** (채팅, 메모리, 스케줄러, 스웜, 설정)
- 컨텍스트 메뉴 통합
- 키보드 단축키

#### 기본 AI 채팅

- AWS Bedrock Claude 연동
- 텍스트 대화
- 대화 기록 저장
- 마크다운 렌더링

#### Claude 모델 지원

- Claude Sonnet 4.6
- Claude Opus 4.6
- Claude Haiku 4.5

#### 대화 메모리 (v2에서 제거됨)

- **CLAUDE.md 스타일 메모리** 시스템
- 대화별 마크다운 메모리 관리
- AI 시스템 프롬프트에 메모리 자동 주입
- 메모리 수동 편집 및 관리

#### 작업 스케줄러 (v2에서 제거됨)

- **chrome.alarms API** 기반 스케줄링
- 3가지 스케줄 타입: **인터벌**, **매일**, **평일**
- 백그라운드 작업 실행
- 스케줄 작업 관리 UI

#### 에이전트 스웜 (v2에서 제거됨)

- 병렬 에이전트 실행 시스템
- **리서치** 및 **토론** 프리셋
- 오케스트레이터가 여러 에이전트 응답 종합
- 에이전트 간 작업 분배

#### FAB (Floating Action Button) (v2에서 제거됨)

- 모든 웹페이지 우하단 **"H" 버튼**
- 5가지 컨텍스트 메뉴 액션
- 빠른 AI 기능 접근

#### 팝업 미니 대시보드 (v2에서 제거됨)

- API 연결 상태 표시
- 활성 작업 목록
- 스토리지 사용량 확인
- 빠른 설정 액세스

#### 페이지 요약

- 현재 페이지 빠른 요약
- 단축키 지원 (Ctrl+Shift+S)
- 주요 내용 추출

#### 텍스트 선택 기능

- 선택한 텍스트 설명
- 텍스트 번역
- 문법 교정

### 🏗️ 초기 아키텍처

- TypeScript + React
- Chrome Extension APIs (Manifest V3)
- AWS Bedrock 직접 통합 (SigV4 서명)
- chrome.storage.local
- chrome.alarms (스케줄러)
- Background Service Worker

### 📊 프로젝트 규모

- **소스 파일**: 20개
- **코드 라인**: 3,500+
- **기능**: 11개
- **AI 모델**: Claude 3개 (Sonnet 4.6, Opus 4.6, Haiku 4.5)
