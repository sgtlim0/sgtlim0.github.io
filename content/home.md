# Welcome to My Wiki

This is a Notion-style wiki built with Next.js and Markdown. You can easily create and organize your knowledge base using simple markdown files.

## Features

- **Markdown Support**: Write content in markdown format
- **Clean UI**: Notion-inspired minimalist design
- **Easy Navigation**: Browse through pages with a sidebar
- **Static Export**: Deploy to GitHub Pages
- **Fast Performance**: Built with Next.js for optimal speed

## Getting Started

1. Edit markdown files in the `content/` directory
2. Create new pages by adding `.md` files
3. Organize content with folders
4. Deploy to GitHub Pages with one command
