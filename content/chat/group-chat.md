# 그룹 채팅

그룹 채팅은 H Chat v3에서 도입된 크로스 프로바이더 비교 기능으로, 여러 AI 모델의 응답을 동시에 확인하고 비교할 수 있습니다.

## 주요 기능

### 크로스 프로바이더 비교

v3에서는 Claude, GPT, Gemini 등 서로 다른 프로바이더의 AI 모델을 동시에 사용할 수 있습니다. 같은 질문에 대해 각 모델이 어떻게 답변하는지 직접 비교할 수 있습니다.

### 단일 입력, 병렬 전송

하나의 질문을 입력하면 선택한 모든 모델에 동시에 전송됩니다. 각 모델에 일일이 질문을 반복할 필요가 없어 시간을 절약할 수 있습니다.

### 모델별 독립 스트리밍

각 모델의 응답은 독립적으로 스트리밍됩니다. 한 모델이 느리더라도 다른 모델의 응답을 실시간으로 확인할 수 있습니다.

### 모델별 응답 시간 표시

각 모델의 응답 생성 시간을 측정하여 표시합니다. 모델 간 속도 차이를 객관적으로 비교할 수 있습니다.

### 프로바이더 간 성능 비교

다양한 측면에서 프로바이더 간 성능을 비교할 수 있습니다:

- **응답 속도**: 어떤 모델이 가장 빠른가?
- **응답 품질**: 어떤 모델의 답변이 가장 유용한가?
- **정확성**: 사실 확인이 필요한 질문에서 어떤 모델이 정확한가?
- **창의성**: 창의적인 작업에서 어떤 모델이 뛰어난가?

## 사용 사례

### 1. 최적의 모델 선택

새로운 프로젝트를 시작할 때, 작업 유형에 가장 적합한 모델을 찾기 위해 여러 모델을 비교합니다.

```
예시: "Python으로 웹 스크래퍼를 작성하는 방법을 알려주세요"
→ Claude, GPT-4, Gemini의 코드 예제와 설명을 동시에 비교
```

### 2. 답변 품질 검증

중요한 의사결정이나 복잡한 문제에 대해 여러 모델의 의견을 종합적으로 검토합니다.

```
예시: "이 비즈니스 전략의 잠재적 위험 요소는 무엇인가요?"
→ 각 모델의 분석을 비교하여 더 포괄적인 인사이트 확보
```

### 3. 모델별 강점 파악

각 모델의 특화 영역을 파악하여 향후 작업에 활용합니다.

```
예시: 코드 생성, 문서 작성, 창의적 글쓰기 등
각 영역에서 어떤 모델이 더 나은 결과를 제공하는지 확인
```

## 사용 방법

1. 채팅 입력창 상단에서 "그룹 채팅" 모드를 선택합니다
2. 비교하고 싶은 AI 모델들을 선택합니다 (최소 2개 이상)
3. 질문을 입력하고 전송합니다
4. 각 모델의 응답이 독립적으로 스트리밍되어 표시됩니다
5. 응답 시간과 내용을 비교하여 최적의 답변을 선택합니다

## 팁

- **일반적인 질문**: 3-4개 모델을 동시에 비교하면 다양한 관점을 얻을 수 있습니다
- **전문적인 질문**: 특정 분야에 강한 모델 2-3개를 선택하여 집중 비교합니다
- **속도 중요 시**: 응답 시간 지표를 참고하여 가장 빠른 모델을 선택합니다
- **품질 중요 시**: 여유를 갖고 모든 모델의 응답을 꼼꼼히 검토합니다
