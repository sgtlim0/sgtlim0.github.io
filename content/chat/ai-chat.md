# AI 채팅

H Chat의 핵심 기능인 AI 채팅은 Claude, GPT, Gemini 등 다양한 AI 모델과 실시간으로 대화할 수 있는 인터페이스를 제공합니다.

## 주요 기능

### 실시간 스트리밍 응답

모든 프로바이더(Claude, GPT, Gemini)에서 실시간 스트리밍 응답을 지원합니다. AI의 답변이 생성되는 즉시 확인할 수 있어 빠른 피드백을 받을 수 있습니다.

### 이미지 업로드 및 Vision 지원

Claude와 GPT 모델에서 이미지 업로드를 지원합니다. 이미지를 업로드하여 Vision 모델을 통해 이미지 분석, 설명, OCR 등의 작업을 수행할 수 있습니다.

### 메시지 액션

각 메시지에서 다양한 액션을 수행할 수 있습니다:

- **복사**: 메시지 내용을 클립보드에 복사
- **핀**: 중요한 메시지를 상단에 고정
- **편집**: 이전 메시지를 수정하여 대화 다시 시작
- **재생성**: 같은 프롬프트로 새로운 응답 생성
- **포크**: 특정 메시지 시점에서 새로운 대화 분기 생성

### 에이전트 모드

에이전트 모드를 활성화하면 AI가 도구를 호출하는 과정을 시각화하여 확인할 수 있습니다. 웹 검색, 페이지 읽기 등의 도구 사용 내역을 단계별로 추적할 수 있습니다.

### 웹 검색 소스 표시

웹 검색 도구를 사용한 경우, AI가 참조한 웹 페이지의 소스를 함께 표시합니다. 정보의 출처를 명확히 확인할 수 있어 신뢰성을 높입니다.

### 내보내기 메뉴

대화 내용을 다양한 형식으로 내보낼 수 있습니다:

- **Markdown**: 마크다운 형식으로 내보내기
- **HTML**: HTML 문서로 변환
- **JSON**: 구조화된 JSON 데이터
- **TXT**: 순수 텍스트 파일

### TTS/STT 기능

- **TTS (Text-to-Speech)**: AI의 응답을 음성으로 들을 수 있습니다
- **STT (Speech-to-Text)**: 음성으로 질문을 입력할 수 있습니다

### 페이지 컨텍스트 인디케이터

현재 대화에 추가된 페이지 컨텍스트를 시각적으로 표시합니다. 어떤 웹 페이지나 문서가 대화의 맥락으로 포함되어 있는지 한눈에 확인할 수 있습니다.

### 프롬프트 라이브러리 연동

자주 사용하는 프롬프트를 라이브러리에 저장하고 불러올 수 있습니다. 반복적인 작업을 효율적으로 처리할 수 있습니다.

### 프롬프트 단축키

채팅 입력창에서 `/`를 입력하면 프롬프트 검색 팝업이 나타납니다. 화살표 키로 이동하고 Enter 또는 Tab으로 프롬프트를 적용할 수 있습니다.

**기본 제공 프롬프트 (8개)**:
- 페이지 요약
- 번역
- 문장 다듬기
- 코드 리뷰
- 이메일 작성
- YouTube 요약
- 논거 분석
- ELI5 (5살 아이에게 설명하듯이)

### 코드 블록 렌더링

AI 응답의 코드 블록은 문법 강조(syntax highlighting)를 지원합니다.

**지원 언어**: JavaScript, Python, HTML, JSON, SQL, C, Go, CSS

각 코드 블록에는 복사 버튼이 포함되어 있어 클릭 한 번으로 코드를 클립보드에 복사할 수 있습니다.

### 빈 대화 제안 카드

새 대화를 시작할 때 4개의 제안 카드가 표시됩니다:
- **페이지 요약**: 현재 페이지 내용 요약하기
- **이메일 작성**: 이메일 초안 생성하기
- **브레인스토밍**: 아이디어 브레인스토밍
- **번역**: 텍스트 번역하기

### 키보드 단축키

채팅에서 사용할 수 있는 키보드 단축키:
- **Ctrl+N**: 새 대화 시작
- **/**: 입력창 포커스 및 프롬프트 검색
- **Escape**: AI 응답 중지
- **Ctrl+K**: 대화 검색

### 대화 포크

대화의 특정 시점에서 새로운 대화를 분기할 수 있습니다. 다양한 대화 방향을 탐색하거나 여러 가능성을 동시에 테스트할 때 유용합니다.

## 페르소나 시스템

6가지 내장 페르소나와 커스텀 페르소나 생성 기능을 제공하여 AI의 대화 스타일을 조정할 수 있습니다:

| 페르소나 | 설명 |
|---------|------|
| **기본 어시스턴트** | 균형 잡힌 일반적인 대화 스타일 |
| **개발자 도우미** | 코드 및 기술 지원에 특화 |
| **작문 코치** | 글쓰기 개선 및 편집 중심 |
| **통역사** | 다국어 번역 전문 |
| **데이터 분석가** | 데이터 분석 및 시각화 지원 |
| **튜터** | 교육적 설명 중심의 답변 |

**커스텀 페르소나**: 아이콘, 이름, 설명, 시스템 프롬프트를 직접 입력하여 자신만의 페르소나를 생성할 수 있습니다.

## 사용 방법

1. 채팅 입력창에 질문이나 요청을 입력합니다
2. 필요한 경우 이미지를 업로드하거나 페이지 컨텍스트를 추가합니다
3. 원하는 페르소나를 선택합니다 (선택사항)
4. 전송 버튼을 클릭하면 실시간으로 AI의 응답을 확인할 수 있습니다
5. 응답에 대해 다양한 액션(복사, 핀, 편집, 재생성, 포크)을 수행할 수 있습니다
