# 변경 이력

## v4.5.0 (2026년 3월) - 통합 AI 워크플로우 플랫폼

H Chat v4의 최신 릴리스로, 배치 OCR, 문서 번역, 템플릿 문서 생성 등 전문적인 AI 도구를 통합한 플랫폼으로 진화했습니다.

### 🎯 신규 기능

#### 배치 OCR

- 최대 10개 이미지 동시 처리
- 4가지 특화 모드: 일반 텍스트, 표/데이터, 수식/기호, 손글씨
- 다국어 지원 (50개 이상 언어)
- 실시간 진행 상황 표시

#### 문서 번역

- TXT/CSV/XLSX/PPTX/PDF 포맷 유지 번역
- 50개 이상 언어 지원
- 원본 레이아웃 및 서식 보존
- 번역 전/후 미리보기

#### 템플릿 문서 생성

- DOCX 템플릿에서 필드 자동 추출
- AI 기반 필드 자동 작성
- 맞춤형 문서 생성
- 템플릿 라이브러리 지원

#### 에이전트 시스템 강화

- 멀티턴 최대 10단계 실행
- 8개 내장 도구:
  - 웹 검색
  - 페이지 읽기
  - URL 가져오기
  - 수학 계산
  - 타임스탬프 변환
  - 텍스트 번역
  - 콘텐츠 요약
  - 북마크/하이라이트 조회
- 도구 실행 결과 실시간 추적

#### 커스텀 플러그인

- Webhook 통합
- JavaScript 실행 환경
- 프롬프트 템플릿 커스터마이징
- 플러그인 마켓플레이스

#### 16개 AI 도구 통합

1. 페이지 요약
2. 멀티탭 요약
3. YouTube 분석
4. 다국어 번역
5. 문법 교정
6. OCR
7. PDF 채팅
8. 데이터 분석
9. 심층 리서치
10. 문서 번역
11. 문서 작성
12. 이미지 생성 (DALL-E 3)
13. 템플릿 문서
14. 차트 데이터 추출
15. 인사이트 보고서
16. 댓글 분석기

#### 심층 리서치 모드

- 비동기 파이프라인 아키텍처
- 실시간 진행 표시
- 다단계 웹 검색 및 분석
- 출처 자동 수집 및 정리

#### 오프라인 메시지 큐잉

- 네트워크 연결 끊김 시 메시지 큐 저장
- 재연결 시 자동 재전송
- 큐 상태 시각화

#### 음성 대화 모드

- STT (Speech-to-Text) 실시간 변환
- 자동 TTS (Text-to-Speech) 재생
- 연속 대화 모드
- 다국어 음성 지원

#### 프롬프트 라이브러리 강화

- "/" 명령어로 빠른 접근
- 100개 이상 기본 프롬프트
- 카테고리별 필터링 (글쓰기/코딩/분석/비즈니스/교육 등)
- 커스텀 프롬프트 변수 지원

#### 커스텀 어시스턴트 빌더

- 특화된 AI 어시스턴트 생성
- 시스템 프롬프트 커스터마이징
- 어시스턴트별 설정 관리
- 어시스턴트 공유

#### 지식 관리

- 스마트 북마크: URL + 요약 + 태그

- 계층적 태그 시스템
- 지식 베이스 검색

#### 전문 메시지 검색

- 1000개 대화에서 100ms 이하 검색
- 트라이그램 인덱싱 + 역인덱싱
- 전문 검색 + 필터링
- 검색 결과 하이라이팅

#### 사고 깊이 제어

- 추론 강도 5단계 조절
- 빠른 응답 vs 심층 분석 선택
- 작업별 최적 깊이 자동 제안

#### 다국어 지원

- 한국어, 영어, 일본어 완벽 지원
- 언어당 600개 이상 번역 키
- 자동 언어 감지
- UI 언어 전환

#### 라이브 데모

- https://hchat-v2-extension.vercel.app 에서 웹 버전 체험 가능

### 📊 프로젝트 규모

- **테스트**: 649개 테스트 케이스
