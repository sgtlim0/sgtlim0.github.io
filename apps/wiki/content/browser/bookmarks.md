# 스마트 북마크

H Chat의 스마트 북마크 기능은 웹페이지의 중요한 텍스트를 하이라이트하고, AI 요약과 함께 저장하여 나중에 쉽게 찾을 수 있도록 합니다.

## 주요 기능

### 텍스트 하이라이팅
- **5가지 색상**: 노랑, 초록, 파랑, 핑크, 보라
- 텍스트 선택 시 플로팅 툴바로 색상 선택
- 하이라이트된 텍스트는 DOM에 `<mark>` 태그로 삽입

### XPath 기반 영속화
- XPath를 사용하여 하이라이트 위치를 정확하게 저장
- 페이지 새로고침 후에도 하이라이트 자동 복원
- DOM 구조가 변경되어도 텍스트 내용 기반으로 복원 시도

### AI 요약 자동 생성
- 하이라이트한 텍스트를 자동으로 AI가 요약
- 긴 텍스트도 핵심만 간결하게 정리
- 나중에 빠르게 내용을 파악할 수 있음

### 고급 관리 기능
- **태그 시스템**: 북마크를 태그로 분류하고 관리
- **전체 검색**: 모든 북마크를 검색하여 빠르게 찾기
- **메모 추가/편집**: 각 북마크에 개인 메모 추가
- **전용 BookmarksView 탭**: 모든 북마크를 한눈에 보기

## 하이라이트 플로우

1. **텍스트 선택**: 웹페이지에서 중요한 텍스트를 드래그하여 선택
2. **플로팅 툴바 클릭**: 나타나는 툴바에서 원하는 색상 선택
3. **DOM 삽입 + 저장**:
   - 선택한 텍스트 주위에 `<mark>` 태그가 삽입됩니다
   - XPath 위치 정보가 Background 스크립트로 전송되어 저장됩니다
4. **페이지 로드 시 복원**:
   - 같은 페이지를 다시 방문하면 저장된 XPath를 사용하여 하이라이트를 자동으로 복원합니다

## 사용 방법

### 하이라이트 추가
1. 웹페이지에서 텍스트를 선택합니다
2. 나타나는 플로팅 툴바에서 색상을 선택합니다
3. 텍스트가 선택한 색상으로 하이라이트됩니다

### 북마크 보기
1. H Chat 확장 프로그램 아이콘을 클릭합니다
2. "북마크" 탭으로 이동합니다
3. 모든 저장된 북마크와 AI 요약을 확인합니다

### 북마크 검색
1. 북마크 탭에서 검색창에 키워드를 입력합니다
2. 텍스트 내용, 요약, 메모, 태그에서 검색됩니다
3. 검색 결과를 클릭하여 원본 페이지로 이동합니다

### 메모 및 태그 추가
1. 북마크 목록에서 북마크를 선택합니다
2. 메모 필드에 개인 노트를 작성합니다
3. 태그를 추가하여 카테고리별로 분류합니다

## 기술 세부사항

### XPath란?
XPath는 XML 문서의 특정 노드를 가리키는 경로 표현식입니다. H Chat은 XPath를 사용하여 웹페이지에서 하이라이트된 텍스트의 정확한 위치를 저장합니다.

### 복원 메커니즘
1. **XPath 매칭**: 저장된 XPath로 노드를 찾습니다
2. **텍스트 검증**: 찾은 노드의 텍스트가 저장된 텍스트와 일치하는지 확인합니다
3. **Fallback**: XPath로 찾지 못하면 텍스트 내용으로 검색을 시도합니다

## 주의사항

- 동적으로 생성되는 콘텐츠(무한 스크롤 등)는 복원이 제한될 수 있습니다
- 페이지 구조가 크게 변경되면 하이라이트가 복원되지 않을 수 있습니다
- 로그인이 필요한 페이지는 로그인 상태를 유지해야 북마크에 접근할 수 있습니다
