# 템플릿 문서 생성

DOCX 템플릿에서 필드를 자동 추출하고 AI가 내용을 채워 넣는 문서 생성 기능입니다.

## 작동 방식

1. DOCX 템플릿 업로드
2. 템플릿 내 플레이스홀더 필드 자동 감지
3. 각 필드에 대한 설명 또는 지시사항 입력
4. AI가 컨텍스트를 고려하여 각 필드 자동 작성
5. 완성된 DOCX 파일 다운로드

## 지원 기능

- 자동 필드 감지 ({{필드명}} 패턴)
- AI 컨텍스트 기반 내용 생성
- 원본 서식/스타일 유지
- 다국어 템플릿 지원
