# 문서 번역

H Chat v4의 문서 번역 기능은 원본 포맷을 유지하면서 50개 이상의 언어로 번역합니다.

## 지원 파일 형식

| 형식 | 포맷 유지 | 설명 |
|------|----------|------|
| TXT | - | 일반 텍스트 번역 |
| CSV | O | 셀 구조 유지, 헤더 보존 |
| XLSX | O | 시트/셀/서식 유지 번역 |
| PPTX | O | 슬라이드 레이아웃 보존 |
| PDF | O | 페이지별 번역 (텍스트 추출 기반) |

## 사용 방법

1. 도구 패널에서 **문서 번역** 선택
2. 소스 언어 / 타겟 언어 설정
3. 파일 업로드
4. **번역 시작** 클릭
5. 번역 진행률 확인
6. 완료 후 번역된 파일 다운로드

## 주요 언어

한국어, 영어, 일본어, 중국어(간체/번체), 프랑스어, 독일어, 스페인어, 포르투갈어, 러시아어, 아랍어, 힌디어, 태국어, 베트남어 등 50개+
