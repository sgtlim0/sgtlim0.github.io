# 웹 검색 + RAG

실시간 웹 검색 결과를 활용하여 최신 정보 기반의 정확한 답변을 제공합니다.

## 주요 기능

### 자동 검색 감지

질문 패턴을 분석하여 웹 검색이 필요한지 자동으로 판단합니다.

**검색이 필요한 질문 예시:**
- "오늘 날씨는?"
- "최신 뉴스 알려줘"
- "2025년 트렌드는?"
- "현재 환율은?"

**검색이 불필요한 질문 예시:**
- "안녕하세요"
- "코드 리뷰 해주세요"
- "이 에러를 수정해주세요"

### 이중 검색 엔진

두 가지 검색 엔진을 지원하며 상황에 맞게 선택됩니다.

#### DuckDuckGo (기본)
- **API 키 불필요**: 설정 없이 즉시 사용 가능
- **HTML 파싱**: 검색 결과 페이지를 직접 파싱
- **프라이버시**: 검색 기록 추적 없음

#### Google Custom Search (선택)
- **API 키 설정**: 설정 페이지에서 API 키 입력
- **높은 정확도**: Google 검색 품질
- **할당량**: 일일 100건 무료 검색

### RAG (Retrieval-Augmented Generation)

검색 결과를 AI 프롬프트에 주입하여 답변 품질을 향상시킵니다.

**동작 방식:**
1. 사용자 질문 분석 → 검색 필요 여부 판단
2. 웹 검색 실행 → 상위 결과 추출
3. 검색 결과를 시스템 프롬프트에 주입
4. AI가 검색 결과 기반으로 답변 생성

**장점:**
- 최신 정보 반영
- 환각(hallucination) 감소
- 출처 기반 답변

### 결과 캐싱

동일한 검색 쿼리를 1시간 동안 캐싱하여 성능을 최적화합니다.

**특징:**
- 쿼리별 캐시 관리
- 1시간 자동 만료
- API 호출 절약

## 사용 방법

### 기본 사용 (DuckDuckGo)

설정 없이 바로 사용 가능합니다.

```
Q: 2025년 AI 트렌드는?
A: [웹 검색 수행] 2025년 주요 AI 트렌드는...
```

### Google Search 설정

더 높은 검색 품질을 원한다면 Google Custom Search API를 설정하세요.

1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. Custom Search API 활성화
3. API 키 생성
4. H Chat 설정 → 웹 검색 → API 키 입력

## 활용 사례

### 최신 정보 질의
```
Q: 오늘 코스피 지수는?
A: [웹 검색] 2025년 3월 2일 코스피는...
```

### 뉴스 및 이벤트
```
Q: 최근 기술 뉴스 요약해줘
A: [웹 검색] 최근 주요 기술 뉴스는...
```

### 비교 및 리뷰
```
Q: iPhone 16 vs Galaxy S25 비교
A: [웹 검색] 두 제품의 주요 차이점은...
```

### 실시간 데이터
```
Q: 서울 날씨 알려줘
A: [웹 검색] 현재 서울 날씨는...
```

## 기술 구현

- **검색 실행**: `src/lib/webSearch.ts`
- **의도 분석**: `src/lib/searchIntent.ts`
- **캐싱**: 메모리 기반 1시간 캐시

## 제한사항

- **DuckDuckGo**: 가끔 파싱 실패 가능 (HTML 구조 변경 시)
- **Google Search**: 일일 100건 무료 할당량 (초과 시 유료)
- **캐시**: 브라우저 재시작 시 캐시 초기화
- **속도**: 검색 수행으로 응답 시간 1-3초 추가

## 팁

- **구체적 질문**: "2025년 3월 2일 환율" 처럼 구체적으로 질문
- **키워드 최적화**: 검색 엔진이 잘 이해할 수 있는 키워드 사용
- **출처 확인**: AI 답변의 출처 링크 확인 권장
- **프라이버시**: 민감한 정보는 검색하지 않기 (로그 가능성)
