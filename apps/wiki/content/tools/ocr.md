# 배치 OCR

H Chat v4의 배치 OCR 기능으로 이미지에서 텍스트를 추출할 수 있습니다.

## 지원 모드

| 모드 | 설명 |
|------|------|
| 일반 텍스트 | 이미지 내 텍스트를 그대로 추출 |
| 표/테이블 | 표 구조를 인식하여 데이터 추출 |
| 명함 | 이름, 직함, 연락처 등 구조화 추출 |
| 영수증 | 항목, 금액, 합계 등 자동 분류 |

## 사용 방법

1. 도구 패널에서 **OCR** 선택
2. 이미지 파일 업로드 (최대 10개)
3. 인식 모드 선택
4. **시작** 클릭

## 지원 형식

- PNG, JPG, JPEG, GIF, WebP, BMP
- 최대 파일 크기: 10MB

## 배치 처리

여러 이미지를 동시에 처리하여 효율을 높입니다:
- 최대 10개 이미지 동시 업로드
- 개별 진행률 표시
- 결과를 한 번에 복사 또는 내보내기
