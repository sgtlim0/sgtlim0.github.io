# 심층 리서치

H Chat v4의 심층 리서치 모드는 복잡한 주제에 대해 자동으로 웹을 검색하고 분석하여 종합 보고서를 생성합니다.

## 작동 방식

1. **주제 분석**: 사용자 질문을 분석하여 검색 전략 수립
2. **웹 검색**: DuckDuckGo, Google, Bing을 통한 다중 소스 검색
3. **페이지 분석**: 검색 결과 페이지의 핵심 내용 추출
4. **종합 분석**: 수집된 정보를 AI가 종합하여 보고서 작성
5. **출처 제공**: 모든 인용에 원본 링크 포함

## 실시간 진행 표시

- 검색 단계별 진행률 표시
- 현재 분석 중인 소스 표시
- 예상 완료 시간 제공

## 사용 사례

- 기술 조사 및 비교 분석
- 시장 동향 리서치
- 경쟁사 분석
- 학술 자료 조사
