# PDF 채팅

PDF 문서를 업로드하고 AI와 대화하며 내용을 탐색할 수 있습니다.

## 주요 기능

### 문서 기반 질의응답

PDF 내용을 기반으로 AI가 질문에 답변합니다.

**특징:**
- **정확한 답변**: PDF 텍스트를 직접 참조하여 답변
- **페이지 추적**: 답변 근거가 되는 페이지 번호 제공
- **컨텍스트 유지**: 대화 맥락을 이해하며 연속 질문 지원
- **클라이언트 전용**: 서버 업로드 없이 브라우저에서 처리 (보안)

### 지원 기능

- **전체 텍스트 추출**: pdfjs-dist 라이브러리 사용
- **페이지별 컨텍스트**: 특정 페이지 내용 참조 가능
- **다국어 지원**: 한국어, 영어 등 다양한 언어 PDF 처리
- **대용량 문서**: 수백 페이지 PDF도 처리 가능

## 사용 방법

1. H Chat 확장 열기
2. 도구 패널 → PDF 채팅 선택
3. PDF 파일 업로드 (드래그 앤 드롭 또는 파일 선택)
4. 문서 로딩 완료 후 질문 입력

## 활용 사례

### 학술 논문 분석
```
Q: 이 논문의 핵심 기여는 무엇인가요?
A: 이 논문은 3가지 주요 기여를 제시합니다 (p.3 참조)...
```

### 계약서/법률 문서 검토
```
Q: 계약 해지 조건이 어떻게 되나요?
A: 제7조에 따르면 계약 해지는... (p.15 참조)
```

### 매뉴얼/기술 문서
```
Q: 이 제품의 설치 방법을 알려주세요
A: 설치는 3단계로 진행됩니다 (p.8-10)...
```

### 보고서 요약
```
Q: 주요 발견사항을 요약해주세요
A: 본 보고서의 핵심 발견사항은...
```

## 기술 구현

- **PDF 파싱**: `src/lib/pdfParser.ts`
- **라이브러리**: pdfjs-dist
- **처리 방식**: 클라이언트 사이드 전용 (서버 업로드 없음)

## 보안 및 프라이버시

- **로컬 처리**: PDF 파일이 서버에 업로드되지 않음
- **세션 메모리**: 브라우저 세션에만 텍스트 저장
- **자동 삭제**: 탭 닫으면 데이터 자동 삭제

## 제한사항

- 이미지 기반 PDF는 OCR 필요 (텍스트 추출 불가)
- 암호화된 PDF는 지원하지 않음
- 매우 큰 파일(100MB+)은 처리 시간이 길 수 있음
- 복잡한 레이아웃(표, 다단 등)은 텍스트 순서가 부정확할 수 있음

## 팁

- **구체적 질문**: "3장의 내용을 요약해주세요" 처럼 구체적으로 질문
- **페이지 지정**: "15페이지에 나온 그래프를 설명해주세요"
- **비교 질문**: "A와 B의 차이점을 설명해주세요"
- **추출 요청**: "핵심 키워드를 나열해주세요"
