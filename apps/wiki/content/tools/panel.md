# 도구 패널

H Chat v3의 도구 패널은 7가지 AI 기반 도구를 통합 인터페이스로 제공합니다.

## 제공 도구

| 도구 | 기능 |
|------|------|
| 📄 페이지 요약 | 현재 탭 내용 추출 + AI 요약 |
| ▶️ YouTube 분석 | 자막/댓글 분석, 통합 리포트 |
| 🌐 번역 | 50개 언어 지원 |
| ✏️ 글쓰기 도구 | 11가지 텍스트 변환 |
| ✅ 문법 교정 | AI 맞춤법/문법 교정 |
| 📸 OCR | 이미지 → 텍스트 추출 (Vision) |
| 📄 PDF 채팅 | PDF 업로드 후 질의응답 |

## 사용 방법

1. 브라우저 우측 상단의 H Chat 확장 아이콘 클릭
2. 도구 패널 탭 선택
3. 원하는 도구 선택 후 사용

## 특징

- **통합 인터페이스**: 모든 도구를 한 곳에서 접근
- **컨텍스트 인식**: 현재 탭, 선택 텍스트, 클립보드 자동 활용
- **빠른 접근**: 사이드바에서 즉시 사용 가능
- **AI 기반**: 각 도구마다 최적화된 AI 프롬프트 적용

## 주요 업데이트

- **v3**: 도구 패널 통합 UI 개선
- **v3**: YouTube 분석, PDF 채팅 신규 추가
