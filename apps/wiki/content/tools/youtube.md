# YouTube 분석

YouTube 영상의 자막과 댓글을 AI로 분석하여 핵심 인사이트를 추출합니다.

## 주요 기능

### 1. 자막 요약

YouTube 영상의 자막을 추출하고 AI로 요약합니다.

**특징:**
- **3단계 Fallback 자막 추출**
  1. `ytInitialPlayerResponse` 객체에서 추출
  2. `ytplayer.config` 백업 추출
  3. HTML 소스 직접 파싱
- **타임스탬프 기반 요약**: 주요 시점별 내용 정리
- **자동 언어 감지**: 한국어/영어 자막 자동 선택

**사용 방법:**
1. YouTube 영상 페이지에서 H Chat 확장 열기
2. 도구 패널 → YouTube 분석 선택
3. "자막 요약" 버튼 클릭

### 2. 댓글 분석

영상 댓글을 감정 분석하고 토픽을 추출합니다.

**분석 내용:**
- **감정 분류**: positive / neutral / negative
- **토픽 추출**: 주요 논의 주제 파악
- **인사이트 도출**: 시청자 반응 종합 분석
- **댓글 수**: 최대 200개 댓글 분석

**사용 방법:**
1. YouTube 영상 페이지에서 H Chat 확장 열기
2. 도구 패널 → YouTube 분석 선택
3. "댓글 분석" 버튼 클릭

### 3. 통합 리포트

자막과 댓글을 종합하여 Markdown 리포트를 생성합니다.

**리포트 구성:**
- 영상 메타데이터 (제목, 채널, 조회수, 좋아요)
- 자막 요약 및 타임스탬프
- 댓글 감정 분포 통계
- 주요 토픽 및 인사이트
- 전체 통합 분석

**사용 방법:**
1. YouTube 영상 페이지에서 H Chat 확장 열기
2. 도구 패널 → YouTube 분석 선택
3. "통합 리포트" 버튼 클릭

## 기술 구현

- **자막 추출**: `src/lib/commentAnalyzer.ts`
- **댓글 분석**: `src/lib/commentAnalyzer.ts`
- **통합 리포트**: `src/lib/insightReport.ts`

## 활용 사례

- 긴 영상 빠르게 파악
- 시청자 반응 모니터링
- 콘텐츠 기획 인사이트 도출
- 학습 자료 요약
- 논쟁적 주제 여론 분석

## 제한사항

- 자막이 없는 영상은 분석 불가
- 댓글이 비활성화된 영상은 댓글 분석 불가
- 최대 200개 댓글까지 분석
