Part 2 — 제품 & 기술

Extension 개발 핵심 주의사항

DOM Race Condition

SPA/Shadow DOM 환경에서 waitForDOMStable() 필수. MutationObserver로 DOM 변경 감지 후 100ms 안정화 대기. SAP Fiori, Jira Cloud 등 SPA 사이트 대응

API 키 Two-Phase 삭제

Extension 제거 시 API 키 즉시 무효화 불가. Phase 1: background SW의 onSuspend에서 서버 무효화 요청. Phase 2: 서버 측 24h TTL 만료로 이중 안전장치

Reflow 최소화

getBoundingClientRect() 호출 배치 처리 필수. Content Script에서 DOM 읽기/쓰기 분리. 테이블 감지 시 IntersectionObserver 우선 사용으로 Layout Thrashing 방지

SSE AbortController

스트리밍 응답 중 사용자 취소/탭 전환 시 AbortController.abort() 필수. Background SW↔Side Panel 간 SSE 연결 라이프사이클 관리. 미처리 시 메모리 누수/좀비 연결 발생
