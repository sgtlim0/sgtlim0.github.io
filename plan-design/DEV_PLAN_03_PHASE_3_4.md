- **주간 실행:** 부하 테스트 + 카오스 테스트 매주 토요일 실행

| 모델 라우팅 실패율 | < 0.1% | 0.1~0.5% | > 0.5% |
| CPU 사용률 | < 70% | 70~85% | > 85% |
| 메모리 사용률 | < 80% | 80~90% | > 90% |

### 6.3 롤백 트리거 및 절차

**자동 롤백 조건 (1개 이상 충족 시):**
- 에러율 > 1.0% 이 5분 이상 지속
- p95 레이턴시 > 3초가 10분 이상 지속
- Self-Healing 루프 무한 반복 감지 (3회 연속 실패)
- Kill Switch 수동 발동

**롤백 절차:**
1. 트래픽을 이전 안정 버전으로 즉시 전환 (< 30초)
2. Canary 인스턴스 격리 및 로그 보존
3. 장애 알림 발송 (Slack #ops-alerts + PagerDuty)
4. 포스트모템 이슈 자동 생성
